## Navigating Directories in Linux

In Linux, folders are called directories. Learning how to move through directories is one of the most important basics.

---

### 📍 pwd (Print Working Directory)

pwd shows the directory you are currently in.

Example:
pwd

---

### 📁 cd (Change Directory)

cd is used to move from one directory to another.

Example:
cd Documents

To go back one step:cd ..

---

### 📂 ls (List)

ls shows all files and directories in your current location.

Example:
ls

---

### 🧭 Linux File System Basics

- / is the root directory (like C:\ in Windows)
- Directories are separated using /
- Windows uses \, Linux uses /

Example path:
/home/user/Documents

---

### ✅ Key Takeaways

- pwd shows where you are
- cd moves you between directories
- ls shows what’s inside a directory
- Linux uses / instead of \
